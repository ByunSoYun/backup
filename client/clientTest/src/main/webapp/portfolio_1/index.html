오류노트 정리

저는 자바 프로그래밍을 하면서 오류메세지가 뜨는것을 오류노트에 정리했습니다.

오류노트에는 해당 오류 메세지가 뜨면 어떻게 해야하는지 해결방안도 써두어서 나중에 똑같은 에러가 생겼을 경우 이 오류 노트를 참고했습니다.

단축키 정리

자바 프로그래밍을 하면서 속도가 중요하다고 생각했습니다.

속도를 높여주는 단축키가 중요하다는 생각을 했고 새로 배우는 단축키가 있으면 단축키 파일에 정리해두었습니다. 이러한 단축키 사용으로 프로그래밍을 보다 편하게 진행 할 수 있었습니다.

Github 사용

저는 수업자료 또는 프로젝트 등 필요한 파일들을 깃허브에 관리합니다.

장소가 바뀌어도 편리하고 다른사람이나 팀원들과 협업 할때 소스코드를 따로 주고받을 필요가 없고 같은 파일을 동시에 열 수 있어서 매우 효율적이기 때문에 저는 Github을 사용하고 있습니다.

Java document 사용
